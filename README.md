# 🎙️ Voice Gender Recognition via Logistic Regression

This project demonstrates a complete Machine Learning pipeline to classify gender based on acoustic properties of the voice.

## 🚀 Key Features
* **Baseline vs. Refined:** Comparative analysis between raw data performance and optimized data.
* **Data Engineering:** Implementation of **IQR (Interquartile Range)** for outlier removal and **Correlation Filtering** for redundancy reduction.
* **Rigorous Validation:** Data split into 70% Train, 20% Validation, and 10% Test to ensure generalization.

## 📊 Results
* **Baseline Accuracy:** ~87%
* **Final Model Accuracy:** ~97%
* **Model:** Logistic Regression with StandardScaler.

## 🛠️ Technologies
* Python, Pandas, Scikit-Learn, Seaborn, Matplotlib.
